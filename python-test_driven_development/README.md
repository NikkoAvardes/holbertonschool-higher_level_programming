# Python – Test-driven development

Projet pour s’entraîner au développement piloté par les tests (TDD) avec des fichiers de tests et doctest.

## Fichiers

- 0-add_integer.py – additionne deux entiers avec validation des types.
- 2-matrix_divided.py – divise tous les éléments d’une matrice par un nombre.
- 3-say_my_name.py – affiche un nom et un prénom formatés.
- 4-print_square.py – affiche un carré à l’aide du caractère « # ».
- 5-text_indentation.py – formate un texte en ajoutant des retours à la ligne après certains caractères.
- 6-max_integer.py – renvoie le plus grand entier d’une liste.

- tests/ – contient les fichiers de tests associés à chaque fonction.
