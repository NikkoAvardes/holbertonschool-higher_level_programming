# JavaScript – Warm up

Petit ensemble de scripts pour se familiariser avec la syntaxe de base de JavaScript et l’exécution avec Node.js.

## Fichiers

- 0-javascript_is_amazing.js – affiche un message simple en JavaScript.
- 1-multi_languages.js – affiche plusieurs lignes dans différentes langues.
- 2-arguments.js – affiche un message selon le nombre d’arguments passés.
- 3-value_argument.js – affiche la première valeur d’argument.
- 4-concat.js – concatène deux arguments avec un texte.
- 5-to_integer.js – convertit un argument en entier et l’affiche.
- 6-multi_languages_loop.js – affiche plusieurs lignes avec une boucle.
- 7-multi_c.js – affiche « C is fun » un certain nombre de fois.
- 8-square.js – affiche un carré avec le caractère « X ».
- 9-add.js – additionne deux entiers passés en arguments.
- 10-factorial.js – calcule la factorielle d’un entier.
- 11-second_biggest.js – affiche le deuxième plus grand entier passé en arguments.
- 12-object.js – manipule un objet JavaScript simple.
- 13-add.js – exporte une fonction qui additionne deux nombres.
- 13-main.js – script de test pour 13-add.js.

- package.json / package-lock.json – configuration du projet Node.js.
