# Python – Input/Output

Projet sur la lecture et l’écriture de fichiers, ainsi que sur la sérialisation en JSON.

## Fichiers principaux

- 0-read_file.py – lit un fichier texte et l’affiche sur la sortie standard.
- 1-write_file.py – écrit une chaîne dans un fichier et retourne le nombre de caractères écrits.
- 2-append_write.py – ajoute du texte à la fin d’un fichier.
- 3-to_json_string.py – convertit un objet Python en chaîne JSON.
- 4-from_json_string.py – convertit une chaîne JSON en objet Python.
- 5-save_to_json_file.py – sérialise un objet dans un fichier JSON.
- 6-load_from_json_file.py – charge un objet depuis un fichier JSON.
- 7-add_item.py – ajoute les arguments de la ligne de commande dans une liste stockée en JSON.
- 8-class_to_json.py – convertit les attributs d’un objet en dictionnaire sérialisable.
- 9-student.py – classe Student avec conversion en JSON.
- 10-student.py – version étendue de Student avec filtrage des attributs.
- 11-student.py – version de Student capable de se recharger depuis un dictionnaire.
- 12-pascal_triangle.py – génère le triangle de Pascal sous forme de liste de listes.

Les fichiers _.json et _.txt présents dans le dossier servent de données ou de fichiers de test.
