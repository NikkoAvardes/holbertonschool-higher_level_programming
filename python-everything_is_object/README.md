# Python - Everything is object

## Description

Ce projet étudie les concepts fondamentaux des objets en Python. Il contient une série d'exercices qui explorent comment Python traite les objets, les références, la mutabilité et l'immutabilité.

Le projet comprend :

- 28 questions théoriques sur le comportement des objets Python (fichiers `*-answer.txt`)
- 1 fonction pratique pour copier une liste (`19-copy_list.py`)

Ces exercices permettent de mieux comprendre :

- La différence entre `==` et `is`
- Les objets mutables vs immutables
- Comment Python gère les références d'objets
- Le comportement des listes, tuples, strings et entiers
