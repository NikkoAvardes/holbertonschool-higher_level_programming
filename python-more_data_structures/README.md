# Python – More data structures: set, dictionary

Scripts pour travailler avec les ensembles (set) et les dictionnaires (dict) en Python.

## Fichiers

- 0-square_matrix_simple.py – applique la fonction puissance 2 à chaque élément d’une matrice.
- 1-search_replace.py – remplace toutes les occurrences d’une valeur dans une liste.
- 2-uniq_add.py – additionne uniquement les valeurs uniques d’une liste.
- 3-common_elements.py – renvoie les éléments communs à deux ensembles.
- 4-only_diff_elements.py – renvoie les éléments présents dans un seul des deux ensembles.
- 5-number_keys.py – retourne le nombre de clés d’un dictionnaire.
- 6-print_sorted_dictionary.py – affiche un dictionnaire trié par clés.
- 7-update_dictionary.py – met à jour ou ajoute une clé dans un dictionnaire.
- 8-simple_delete.py – supprime une clé d’un dictionnaire.
- 9-multiply_by_2.py – multiplie toutes les valeurs d’un dictionnaire par 2.
- 10-best_score.py – retourne la clé ayant la plus grande valeur.
- 11-multiply_list_map.py – multiplie tous les éléments d’une liste avec `map`.
- 12-roman_to_int.py – convertit un nombre romain en entier.
