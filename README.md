# Programmation de plus haut niveau – Holberton School

Ce dépôt regroupe plusieurs projets de programmation de plus haut niveau (Python, JavaScript, SQL, APIs REST) du cursus Holberton School.

## Dossiers principaux

- python-hello_world – premiers scripts Python (affichage, chaînes, nombres).
- python-if_else_loops_functions – conditions, boucles et fonctions en Python.
- python-import_modules – import de modules, utilisation de fonctions externes et arguments de ligne de commande.
- python-data_structures – listes et tuples.
- python-more_data_structures – ensembles (set) et dictionnaires (dict).
- python-everything_is_object – modèle objet de Python (références, mutabilité / immutabilité).
- python-exceptions – gestion des exceptions et erreurs.
- python-classes – classes simples et objets de base (carré).
- python-more_classes – classes et objets avancés (Rectangle, comptage d’instances, etc.).
- python-inheritance – héritage, isinstance/issubclass, géométrie.
- python-input_output – lecture/écriture de fichiers et JSON.
- python-object_relational_mapping – mapping objet-relationnel entre Python et MySQL.
- python-serialization – sérialisation JSON, pickle, CSV, XML.
- python-server_side_rendering – rendu côté serveur avec Python et templates.
- python-test_driven_development – développement piloté par les tests (doctest / unittest).
- python-abc – classes abstraites, duck typing et mixins.
- javascript-warm_up – scripts JavaScript de base (syntaxe, boucles, fonctions).
- javascript-dom_manipulation – manipulation du DOM côté client.
- SQL_introduction – requêtes SQL de base avec MySQL.
- SQL_more_queries – requêtes SQL plus avancées (jointures, privilèges, agrégations).
- restful-api – consommation et exposition d’APIs REST simples.
