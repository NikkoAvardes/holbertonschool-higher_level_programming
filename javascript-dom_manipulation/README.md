# Manipulation du DOM avec JavaScript

Ce projet contient des exercices pratiques pour apprendre la manipulation du DOM en JavaScript. Il comprend 9 exemples progressifs qui montrent les concepts fondamentaux.

## Structure du projet

Le projet est organisé en paires de fichiers :

- `0-main.html` / `0-script.js` - Modification basique de style
- `1-main.html` / `1-script.js` - Gestionnaire d'événement simple
- `2-main.html` / `2-script.js` - Manipulation des classes CSS
- `3-main.html` / `3-script.js` - Toggle de classes
- `4-main.html` / `4-script.js` - Création d'éléments dynamiques
- `5-main.html` / `5-script.js` - Modification du contenu HTML
- `6-main.html` / `6-script.js` - Fetch API avec données externes
- `7-main.html` / `7-script.js` - Liste dynamique via API
- `8-main.html` / `8-script.js` - Chargement asynchrone avec DOMContentLoaded

## Comment utiliser

1. **Ouvrir les exemples**
   - Double-cliquez sur n'importe quel fichier `*-main.html` dans votre navigateur
   - Ou faites clic droit → "Ouvrir avec" → navigateur de votre choix

2. **Explorer le code**
   - Ouvrez les fichiers `.js` correspondants pour voir la logique
   - Modifiez le code et rechargez la page pour voir les changements

3. **Déboguer**
   - Utilisez F12 pour ouvrir les outils de développement
   - Consultez la console pour les erreurs ou messages

## Ce que vous apprendrez

- **Sélection d'éléments** : `querySelector`, `getElementById`
- **Gestion d'événements** : `addEventListener`, `click`, `DOMContentLoaded`
- **Modification du DOM** : `textContent`, `innerHTML`, `style`
- **Classes CSS** : `classList.add()`, `classList.remove()`, `classList.toggle()`
- **Création d'éléments** : `createElement`, `appendChild`
- **API externes** : `fetch()`, gestion des promesses, JSON

## Prérequis

- Un navigateur web moderne
- Un éditeur de texte pour modifier les fichiers JS
- (Optionnel) Node.js pour les outils de formatage

## Formatage du code

Si vous avez Node.js installé, vous pouvez formater le code avec :

```bash
npx semistandard --fix nomDuFichier.js
```
