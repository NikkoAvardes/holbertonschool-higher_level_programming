# RESTful API

Exercices pour consommer et exposer des APIs REST simples en Python.

## Fichiers

- task_02_requests.py – envoie des requêtes HTTP avec la bibliothèque `requests`.
- main_02_requests.py – script de test pour `task_02_requests.py`.
- task_03_http_server.py – implémente un petit serveur HTTP minimaliste.
- task_04_flask.py – API REST basique avec Flask.
- task_05_basic_security.py – exemples de notions de sécurité autour des endpoints (authentification, filtrage basique).

- posts.csv – fichier de données utilisé par certains scripts.
