# Python – Server side rendering

Petit projet d’introduction au rendu côté serveur avec Python et des templates HTML.

## Fichiers

- task_00_intro.py – script introductif montrant un rendu HTML simple.
- task_01_jinja.py – utilise Jinja2 pour générer du HTML à partir de templates.
- task_02_logic.py – ajoute de la logique Python dans les templates.
- task_03_files.py – lit des données depuis des fichiers pour les afficher.
- task_04_db.py – lit des données depuis une base (products.db) pour les afficher.

- templates/ – répertoire contenant les templates HTML.
- items.json, products.csv, products.json, products.db – fichiers de données utilisés par les scripts.
- main*00*.py – script de test / exemple d’exécution.
