# SQL – More queries

Scripts SQL pour pratiquer des requêtes plus avancées : gestion des utilisateurs, contraintes, jointures et regroupements.

## Fichiers

- 0-privileges.sql – affiche les privilèges des utilisateurs MySQL.
- 1-create_user.sql – crée un utilisateur avec des privilèges.
- 2-create_read_user.sql – crée un utilisateur avec des droits en lecture.
- 3-force_name.sql – crée une table avec contrainte NOT NULL sur une colonne.
- 4-never_empty.sql – renforce les contraintes de non-nullité.
- 5-unique_id.sql – crée une table avec clé primaire auto-incrémentée.
- 6-states.sql – insère des lignes dans la table `states`.
- 7-cities.sql – insère des lignes dans la table `cities`.
- 8-cities_of_california_subquery.sql – récupère les villes de Californie (sous-requête).
- 9-cities_by_state_join.sql – affiche les villes avec leur État (JOIN).
- 10-genre_id_by_show.sql – récupère les genres d’une série donnée.
- 11-genre_id_all_shows.sql – genres pour toutes les séries.
- 12-no_genre.sql – séries sans genre associé.
- 13-count_shows_by_genre.sql – compte de séries par genre.
- 14-my_genres.sql – genres d’une série spécifique.
- 15-comedy_only.sql – séries dont le genre est « Comedy ».
- 16-shows_by_genre.sql – séries regroupées par genre.
