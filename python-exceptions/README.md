# Python – Exceptions

Scripts pour apprendre à lever et gérer les exceptions en Python.

## Fichiers

- 0-safe_print_list.py – affiche en sécurité les éléments d’une liste (gère les IndexError).
- 1-safe_print_integer.py – affiche un entier en capturant les erreurs de type.
- 2-safe_print_list_integers.py – affiche uniquement les entiers contenus dans une liste.
- 3-safe_print_division.py – divise deux nombres et retourne le résultat en gérant les erreurs.
- 4-list_division.py – divise élément par élément deux listes.
- 5-raise_exception.py – lève volontairement une exception de type (`TypeError`).
- 6-raise_exception_msg.py – lève une exception (`NameError`) avec un message personnalisé.
